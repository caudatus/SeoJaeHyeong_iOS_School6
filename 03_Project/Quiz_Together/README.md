간단한 OX퀴즈 게임입니다.
OX의 버튼을 탭하여 답을 입력하고 그 결과를 표시하며
또한 진행상황과 점수를 계산하여 표시합니다.

오픈소스 라이브러리 ProgressHUD 를 사용하였습니다.

ProgressHUD is a lightweight and easy-to-use HUD for iOS. (Objective-C)
