# Xcode Shortcuts

## Basic
 key | 설명
---------- | ----------
⌘ + ⇧ + 0	| 개발자 문서
⌘ + ,	| 설정
⎋	| 자동 완성 리스트
⌘ + L	| 지정 라인으로 이동
⌘ + Z	| 되돌리기 (Undo)
⌘ + ⇧ + Z	|	다시하기 (Redo)
⌃ + I	|	들여쓰기 조정
⌘ + N	|	새 파일 만들기
⌘ + ⇧ + N	|	새 프로젝트 만들기
⌘ + Q	|	Xcode 종료
⌘ + /	|	주석 처리 / 해제
---
## Assistant / Version Editor
key | 설명
---------- | ----------
⌘ + ⌥ + ↩ |	보조 에디터 열기
⌘ + ⌥ + ⇧ + ↩ |	버전 에디터 열기
⌘ + ↩ |	기본 에디터 모드 (보조 / 버전 에디터 닫기)
⌘ + ⇧ + O |   이후   ⌥ + ↩	지정 파일 보조 에디터로 열기
⌃ + ` |	다음 에디터로 포커스 이동
⌃ + ⇧ + ` |	이전 에디터로 포커스 이동
⌘ + J |	에디터 포커스 이동 선택 화면

## Delete
key | 설명
---------- | ----------
⌘ + ⌫ |	커서 위치부터 행의 시작까지 삭제
⌘ + fn + ⌫ |	커서 위치부터 행의 끝까지 삭제
⌃ + K |	커서 위치부터 단락(Paragrahp)의 끝까지 삭제
⌥ + ⌫ |	뒤쪽 방향으로 단어 단위 삭제
⌥ + fn + ⌫ |	앞쪽 방향으로 단어 단위 삭제
⌃ + ⌫ |	뒤쪽 방향으로 subword  단위 삭제 (대소문자로 구분)
⌃ + fn + ⌫ | 앞쪽 방향으로 subword  단위 삭제 (대소문자로 구분)

## Text
key | 설명
---------- | ----------
⌘ + F |	현재 파일 내 검색
⌘ + ⇧ + F |	지정 범위 내 검색 (기본값 : 프로젝트 전체)
⌘ + ⌥ + F |	현재 파일 내 치환
⌘ + ⌥ + ⇧ + F |	지정 범위 내 치환 (기본값 : 프로젝트 전체)
⌘ + ⌥ + [  /  ] |	위 / 아래 방향 라인 스위칭
⌘ + E |	현재 선택된 텍스트 블럭을 검색용 텍스트로 지정 (⌘ + F
⌘ + G |	현재 파일에서 검색용 텍스트와 일치하는 다음 문자열로 이동
⌘ + ⇧ + G |	현재 파일에서 검색용 텍스트와 일치하는 이전 문자열로 이동
⌘ + ⌃ + E |	범위 내 동일 키워드(변수 등) 치환 (Edit All in Scope)
⌃ + / |	다음 플레이스홀더로 이동
⌃ + ? |	이전 플레이스홀더로 이동

## Debugging
key | 설명
---------- | ----------
⌘ + ‘  /  “ |	다음 / 이전 이슈로 이동
⌘ + ⌥ + ⌃ + F |	수정 가능한 모든 이슈 자동 수정
⌘ + K |	콘솔창 텍스트 Clear
⌘ + \ |	브레이크 포인트 설정 / 해제
⌘ + Y |	브레이크 포인트 전체 활성화 / 비활성화
⌘ + ⌃ + Y |	Pause / Continue
F6 |	Step Over  (다음 라인으로 이동, 메서드 무시)
F7 |	Step Into   (다음 라인으로 이동, 메서드가 있을 경우 내부로 진입)
F8 |	Step Out   (진입한 메서드에서 빠져나옴)

## Storyboard
